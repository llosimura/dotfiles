# Dotfiles

My dotfiles.

## Installation

1. Clone this repository.
2. Run `./script/setup`.

## Folder structure

All the configuration is inside the `config` folder:

```
config
├── bin # Any executable. Everything here makes it to the user path
├── editors
├── git
├── hyper
├── nvim
├── shells
├── spotify
└── tmux
```
